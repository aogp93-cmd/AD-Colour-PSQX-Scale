# AD-Colour-PSQX-Scale
A scale adjuster for Filmmaker V10 to adjust the PSQX Scale per Queue 

Please install the software into the main drive of your computer (Normally C Drive) 
Once installed - Open the software 

DOWNLOAD HERE

How to use: 

Once the program is open - Please make sure you copy of FILMMAKER V10 is not open when making adjustments
Check the Queues route by clicking Browse... - You should see the Qeueue names the same as within Filmmaker V10
If these are not showing, find the Filmmaker V10 Program files, Open, Select Queues, Click select folder 
On the Queue drop down - Slect the Queue in which you would like to make adjustments 
Now click on the file Dropdown - Their should only be 1 file available - Select this file
It should now load the current X and current Y which are already set within Filmmaker V10
Now you need to input your adjustments - Current Size:  This is the current size of the Positive print, For example 599mm x 599mm 
Now you need to input your intended print size - Requested size: This should be the size the Positive should of printed, For example 600mm x 600mm 
Now click Apply & Save, A diaglog box will now open - It will alert you to make sure the FILMMAKER V10 software should be closed, It will show the new Scale % and confirm a back up of the original will be made 
Click OK
Your software has now been updated - You can now open Filmmaker V10 and test and confirm 

Back Up Now: Select this to create a back up of the Queue settings without making adjustments 
Restore Backup - If you have made an error - Please select restore Backup 
Reset to 100% - If you had made a lot of changes and want to start from fresh - Select Reset to 100% - You would need to print a new file after this and then run the software again to make adjustments.
